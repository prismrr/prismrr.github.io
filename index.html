Projetos do Grupo PRISM


Projetos em andamento:

Map2Check

Um método automatizado para verificação de propriedades de segurança de mémoria para programas escritos em C.

Kraken

Sistema autonomo para detecção e estimativa de objetos submersos em água.

GAN-AGE

Um sistema para geração de imagens baseado em previsão temporal para pessoas desaparecidas.

Xray-Vision

Diagnóstico De Pneumonia Na Infância Utilizando Rede Neural Convolucional Profunda Para Reconhecimento De Padrões Em Imagens De Raio X.


Projetos finalizados:

V2C

Um método para verificação formal de programas escritos em VHDL, baseado em transformações de código.

AutomailX

Uma prótese robótica para geração de ações em movimentos baseada em padrões musculares.

Hand.io

Uma luva inteligente para o reconhecimento de padrões em gestos para conexão e comunicação entre dispositivos eletrônicos.
